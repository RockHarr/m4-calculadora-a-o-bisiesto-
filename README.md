Mini app educativa en **JavaScript ES6+ (vanilla)** que pide al usuario **año** y **mes (1–12)** y muestra la **cantidad de días** del mes, considerando **año bisiesto** (regla gregoriana).  
Interfaz de entrada/salida con **SweetAlert2** y estilo base con **Bootstrap 5** usando **CDN** (sin build, sin npm) para que sea liviano en **StackBlitz**.

---

## Objetivo del ejercicio

- El usuario ingresa **mes (1 a 12)**.
- (Extensión) El usuario ingresa **año** para calcular febrero como 28 o 29.
- Usar **arreglos** para almacenar meses y días.
- Aplicar **condicionales** necesarias.
- Validar **todas** las entradas posibles (vacío, no número, fuera de rango, cancelar, etc.).

---

## Tecnologías

- **HTML5**
- **JavaScript ES6+ (Vanilla)**
- **Bootstrap 5 (CDN)**
- **SweetAlert2 (CDN)**
